Tableau Visualizations of Ruoyu Cheng

Below is a collection of projects I have accumulated using data visualization tools (Tableau). The content presented is not only for the techniques of the tool, but also contains the corresponding business logic.

Hospital Visit Data Visualization

Data storytelling is the key to the entire project. And this is built on the foundation of Dashboard. The use of Dashboard presentations is central to the performance of this data analysis. The first Dashboard is a combination of actual and desired revenue comparisons, KPI-donut charts, thermometer char, and highlight charts. Next is the presentation of performance analysis and user analysis. This includes patient satisfaction rates for health care and medical facilities.Specifically for charting techniques many classical graphic representations are used qualitatively and quantitatively, including Gauge Chart, Heat Map, Likert Charts, etc. In the final data story and dashboard, the computational parameters and movable interactions are set to make the whole project more logically rational.

Real Estate/Eurapo Superstone/Wold GHG Emission 1990-2018

The above three datasets are trained and presented accordingly for trend prediction and statistical analysis from various aspects.The application of scatter plots and trend lines and the addition of R-squares give more statistical significance to price trends, making data visualization not a simple expression but a valuable accumulation. Candlestick Chart is so classic that it conveys every financial information about stock price in a delicate way. The same dynamic and interactive representation has been used in Dashboard.Such a presentation would reflect the fluidity and granularity of the data logic.

MLB/Global Superstore/US CO2 Emission 1990-2014

For MLB, the main focus is on the analysis of player performance, and the application of the Measure filter and parameter control allows the data to be expressed in a granular and consistent manner. Waterfall charts are a powerful tool for representing discrete relative quantitative relationships. The Bullet Chart and Bar-in-Bar Chart are excellent tools for measuring purpose and status.And then for the Supperstore analysis, the dual-axis data performed very well. And the strong correlation between sales data and time makes the application of target and benchmarking analysis particularly important in Calender.The grouping and layered presentation of geographic data has always been a strength of Tableau, so a more interesting approach to map legend presentation is half the battle when it comes to geographically differentiated Co2 emissions presentation.

Superstore/MLB/DSNY

In this analysis more emphasis is placed on the basic data visualization process and basic conceptual representation. First is the basic application of text table, then tree map has an important role for quantifying the distribution. Secondly, for the breakdown of income, the application of multiple pie charts can help Ad-hoc analysis; Funnel Chart applies the idea of positive and negative to avoid unreasonable data representation; 100% Bar Chart can show the relative value of data when comparing absolute values; Packed Bubble Chart is certainly practical with strong ornamental features. The Packed Bubble Chart is certainly a useful chart with a strong visual appeal.The application of the classic graphical Gantt chart also makes the task management process more intuitive.
